# aoneng-fe-monitor

> 奥能电源 前端监控稳定性

1. 前端监控 npm 包
   1. [aoneng-fe-monitor-browser](https://www.npmjs.com/package/aoneng-fe-monitor-browser)：前端稳定性监控 页面监控；
   2. [aoneng-fe-monitor-core](https://www.npmjs.com/package/aoneng-fe-monitor-core)：前端稳定性监控 核心功能；
   3. [aoneng-fe-monitor-react](https://www.npmjs.com/package/aoneng-fe-monitor-react)：前端稳定性监控 React 监控；
   4. [aoneng-fe-monitor-vue](https://www.npmjs.com/package/aoneng-fe-monitor-vue)：前端稳定性监控 Vue 监控；
   5. [aoneng-fe-monitor-web](https://www.npmjs.com/package/aoneng-fe-monitor-web)：前端稳定性监控 Web 监控；
   6. [aoneng-fe-monitor-web-performance](https://www.npmjs.com/package/aoneng-fe-monitor-web-performance)：前端稳定性监控 Web 性能监控；
   7. [aoneng-fe-monitor-wx-mini-program](https://www.npmjs.com/package/aoneng-fe-monitor-wx-mini-program)：前端稳定性监控 小程序监控；
   8. [aoneng-fe-monitor-wx-mini-program-performance](https://www.npmjs.com/package/aoneng-fe-monitor-wx-mini-program-performance)：前端稳定性监控 小程序性能监控；
2. 监控异常收集 node 服务；
3. 前端监控异常告警&界面展示；

## 技术选型

- 包管理工具：pnpm；
- 构建工具：encode-bundle；
- 数据清洗&存储：Node 生态；
